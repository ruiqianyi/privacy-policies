# ruiqianyi 通用隐私政策

本目录包含适用于 ruiqianyi 组织下所有本地存储应用的通用隐私政策模板。

## 核心特点

所有 ruiqianyi 应用的共同特点：
- ✅ **完全本地存储** - 所有数据保存在用户设备本地
- ✅ **零网络请求** - 不连接任何外部服务器
- ✅ **无第三方服务** - 不集成任何第三方 SDK
- ✅ **数据完全私密** - 开发者无法访问用户数据
- ✅ **无数据泄露风险** - 不存在网络传输和云端存储

## 可用语言

本隐私政策提供以下语言版本：

- 🇨🇳 **中文** - `PRIVACY_POLICY_GENERIC_ZH.md`
- 🇺🇸 **English** - `PRIVACY_POLICY_GENERIC_EN.md`
- 🇯🇵 **日本語** - `PRIVACY_POLICY_GENERIC_JA.md`
- 🇰🇷 **한국어** - `PRIVACY_POLICY_GENERIC_KO.md`

## 使用方法

### 对于应用开发者

1. 根据应用发布地区选择相应语言的隐私政策
2. 将隐私政策文件复制到应用项目根目录
3. 在应用商店提交时引用该隐私政策
4. 可在应用内提供隐私政策查看入口

### 对于用户

- 这些隐私政策适用于所有 ruiqianyi 开发的本地存储应用
- 您的数据完全保存在您的设备上，我们无法访问
- 您可以随时删除数据或卸载应用
