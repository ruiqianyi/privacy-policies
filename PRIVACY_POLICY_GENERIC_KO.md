# 개인정보 처리방침

**시행일: 2025년 1월 31일**

## 소개

ruiqianyi가 개발한 애플리케이션(이하 "본 앱")을 이용해 주셔서 감사합니다. 저희는 귀하의 개인정보 보호를 매우 중요하게 생각합니다. 본 개인정보 처리방침은 개인정보의 수집, 사용 및 보호 방법에 대해 설명합니다.

## 핵심 원칙

**본 앱은 완전히 로컬화된 데이터 저장 솔루션을 채택합니다**. 모든 데이터는 귀하의 기기에 로컬로 저장되며, 외부 서버나 클라우드 서비스에 업로드되지 않습니다.

## 정보 수집 및 사용

### 수집하는 정보

본 앱은 귀하가 사용 중에 능동적으로 입력하거나 생성한 데이터만을 기기의 로컬에 수집 및 저장합니다. 구체적인 데이터 유형은 앱의 기능 특성에 따라 다릅니다.

### 데이터 저장 방법

**모든 데이터는 귀하의 기기의 로컬 저장소**(SQLite 데이터베이스, 로컬 파일 등)에 저장되며, 네트워크 전송은 전혀 이루어지지 않습니다. 이는 다음을 의미합니다:

- 귀하의 데이터는 완전히 기기에 저장됩니다
- 저희는 귀하의 데이터에 접근할 수 없습니다
- 데이터는 제3자와 공유되지 않습니다
- 네트워크 요청이나 외부 데이터 전송이 전혀 없습니다
- 네트워크를 통한 데이터 유출 위험이 없습니다

### 데이터 사용 목적

수집된 정보는 다음 목적으로만 사용됩니다:

- 앱의 핵심 기능 및 서비스 제공
- 귀하의 개인 설정 및 환경설정 저장
- 앱의 사용자 경험 개선
- 앱 내 데이터 표시 및 관리 기능 구현

## 데이터 보안

모든 데이터가 기기의 로컬에 저장되므로:

- 데이터 보안은 전적으로 귀하의 기기 보안 설정에 따라 달라집니다
- 데이터 손실을 방지하기 위해 정기적으로 기기 데이터를 백업할 것을 권장합니다
- 앱을 제거하면 로컬 데이터가 영구적으로 삭제됩니다
- 클라우드 백업, 동기화 또는 복구 서비스를 제공하지 않습니다
- 화면 잠금 비밀번호 등의 기기 보안 조치를 사용하여 데이터를 보호할 것을 권장합니다

## 권한 안내

본 앱은 다음 기기 권한이 필요할 수 있습니다(앱 기능에 따라 다름):

- **저장소 권한**: 기기의 로컬에 데이터 파일을 읽고 쓰기 위함
- **기타 기능 권한**: 특정 앱 기능을 구현하기 위해서만 사용

저희의 약속:
- 불필요한 권한은 요청하지 않습니다
- 모든 권한은 앱의 핵심 기능에만 사용됩니다
- 권한을 사용하여 앱 기능과 무관한 정보를 수집하지 않습니다

## 제3자 서비스

본 앱은 **어떠한 제3자 서비스도 통합하지 않습니다**. 다음을 포함하되 이에 국한되지 않습니다:

- 제3자 데이터 분석 서비스
- 광고 서비스 또는 광고 SDK
- 소셜 미디어 SDK
- 클라우드 스토리지 서비스
- 사용자 행동 추적 서비스

귀하의 데이터는 제3자와 공유되거나 전송되지 않습니다.

## 네트워크 연결

본 앱은 **어떠한 네트워크 요청도 하지 않습니다**. 다음을 포함합니다:

- 원격 서버에 연결하지 않음
- 사용자 데이터를 업로드하지 않음
- 외부 콘텐츠를 다운로드하지 않음(앱 스토어의 정상적인 업데이트 제외)
- 어떠한 형태의 데이터 동기화도 하지 않음

## 아동 개인정보 보호

본 앱은 13세 미만 아동으로부터 고의로 개인정보를 수집하지 않습니다. 보호자께서 자녀가 동의 없이 본 앱을 사용한 것을 발견하신 경우 저희에게 연락해 주시기 바랍니다.

## 데이터 삭제

다음 방법으로 언제든지 데이터를 삭제할 수 있습니다:

- 앱 내에서 특정 데이터 레코드 삭제
- 앱 데이터 지우기(기기의 앱 관리 기능 사용)
- 앱 제거 시 로컬에 저장된 모든 데이터가 영구적으로 삭제됩니다

## 데이터 내보내기

데이터가 로컬에 저장되므로 다음이 가능합니다:

- 기기의 백업 기능을 사용하여 데이터 내보내기
- 기기의 파일 관리 기능을 사용하여 데이터 파일에 접근(기기가 허용하는 경우)
- 앱이 제공하는 내보내기 기능(사용 가능한 경우)을 사용하여 데이터 내보내기
